关于ubuntu

Posted on 2016-09-11 | In Ubuntu

前段时间装上了Ubuntu,放心我不是来强调他的各种优势的,只是来吐嘈吐嘈

  • 挂机问题
    　某天,电脑开在家里,出去找小伙伴玩了一下午,回来之后想着继续编程,然后我就惊了个呆,当时以为是我的电脑gg了,立马强制关机重启,发现电脑跟没事一样,就此翻过..
    　开学之后,三番两次出现这个问题,没法忽视了,百度了各种方法,也亲自试了,并没有什么X用..

    额还是说说我怎样傻瓜式做法吧:系统设置->电源->全部设置为任何时候都不要挂起
    另一个方法可惜我并没有成功,不想再折腾了

  • 磁盘挂载
    　之前win10下了第三方的谷歌插件,所以换系统之后无法同步到Utuntu下的chrome中,我只好重新回win10把LivePage这个拓展程序打包下载下来,然后把crx文件放到D盘(想着随便放哪里,反正都可以访问) 结果,重启进入Ubuntu之后发现..无法访问 ? ! 除了分给Utuntu的磁盘分区都无法访问

    Error mounting /dev/sdax

    据了解使用ntfs 命令可以修复该问题，操作步骤:

方法一 (这个方法试了之后文件都打不来了 重启之后可以访问了)

(可以先用fdisk -l 查看不能挂载的磁盘分区) 1.安装ntfs sudo apt-get install ntfs-3g 2.修复命令 sudo ntfsfix /dev/sda6 //可以通过双击磁盘来得到例如sda6的分区 也可以用fdisk-l查看 好了 我可以找到我的crx文件 安装插件了hh

方法二(这个留着下次试吧 反正我也不担心他不出问题–) The NTFS partition is in an unsafe state. Please resume and shutdown Windows fully (no hibernation or fast restarting), or mount the volume read-only with the 'ro' mount option.

(NTFS分区处于不安全状态,Win默认开启了快速启动功能,可执行下面的步骤。) 1.打开电源选项，找到并点击左侧的选择电源按钮的功能， 2.向下滚动窗口，可看到关机设置中的启用快速启动 -> 关闭快速重启功能

  • 输入法 想睡觉了,不说了 反正自带的我用着不爽 就不爽

总结:早换mac早超生 - - 等我熟悉linux command line之后吧
