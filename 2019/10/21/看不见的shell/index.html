看不见的shell

Posted on 2019-10-21 | In Ubuntu
ubuntu不好玩 shell好玩

我好难啊，用了八百遍的命令，一段时间不用就丢回计算机了，为什么记忆不能久一点呢…

conda

指令 功能
conda clean -p 删除下载的库安装包
conda clean -t 删除初始的库安装包
conda create -n name python=x 创建名为name的python环境，x为python版本号
conda remove name 删除名为name的python库
conda remove -n name –all 删除名为name的python环境
conda/source activate name 激活名为name的python环境
conda/source deactivate 退出python环境
conda info -e 查看已安装的环境列表

文件操作

文件移动/拷贝

指令 功能
ln -s 源目录 目标目录 创建软链接(如果提示 cd: 符号连接的层数过多 使用绝对路径)
rm 目标目录 删除软链接
mv 源文件(夹) 目标目录 移动文件(夹)
mv 源目录/* 目标目录 移动文件夹下所有文件(这个有点坑，最好是带文件夹一起移动)
cp -r 源目录 目标目录 拷贝文件夹

文件权限

指令 功能
sudo chown -R username:groupname filesname 修改文件夹的用户所有者
sudo chowd rwx files 修改文件夹读写权限
stat filename 查看文件最近修改、创建时间
ls -l filaname 查看文件权限

文件解压/打包 (zsh x)

文件类型 指令 备注
.tar tar -xvf x:extract v:显示执行过程 f:指定文件 t:列出文件内容
.gz gzip -d/gunzip directory c:保留原始文件
.tar.gz/.tgz -C directory tar -xzf tar.gz压缩占用cpu小
.rar unrar x 需要安装unrar
.zip unzip -d directory 用的最多的文档格式，压缩率不高，l:列出文件内容

查找文件

指令 功能
which filename 在环境变量$PATH中查找（寻找可执行文件）
find grep info
find /指定文件夹 -name filename 查找当前目录(包括子目录)下文件名中含fileinfo的文件
ll |grep info 功能同上，但无法递归子目录
find | xargs grep info 查找当前目录(包括子目录)下文件内容中含info的文件
locate filename linux将所有文件存在一个数据库中(一周更新一次，实时性不好)，但效率比find高

系统相关

指令 功能
source ~/.zshrc 激活环境变量
free -m 查看内存使用
nvidia-smi -l 查看gpu状态 -l持续
top -p 查看指定进程号(其实用htop就好)
ps 进程号 查看进程详细状态
df -h 查看文件系统的磁盘空间占用情况
fdisk -l 查看磁盘分区
sudo passwd userName 设置密码
sudo smartctl –all /dev/sda 查看硬盘 /dev/sda的基本信息
sudo useradd -m -s /bin/bash XXX 创建XXX新用户
sudo adduser XXX 傻瓜式创建XXX新用户
sudo userdel -r userName 彻底删除XXX用户
kill PID 杀死进程
lsof -i：9696 查看网络端口使用
iotop 查看io进程，查看磁盘读写速度
halt/poweroff/shltdown -h now 立刻/十分钟后关机
reboot/shutdown -r now 重启系统
sudo vim /etc/sudoers 将用户添加进sudoers(可能需要先修改sudoer权限)

服务器相关

指令 功能
nvcc –version 查看服务器CUDA版本
cat /usr/local/cuda/version.txt 查看服务器CUDA版本-方法二

cat /etc/shells 查看系统shell版本
chsh -s /bin/zsh 切换zsh
nohup python -u xxx.py > result.log 2>&1 & 后台运行python程序
